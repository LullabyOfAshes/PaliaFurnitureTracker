# 🌙 Palia Furniture Tracker

A cozy desktop application built with Python and Tkinter to track furniture collection progress in Palia.

---

## ✨ Features

- 🪑 Track furniture collection progress by set
- 🔍 Search across sets and individual items
- 📊 Live progress tracking (per set + overall)
- ✔ Mark items as collected with a simple click
- 🧹 One-click complete or reset sets
- 💾 Persistent save system (data stored locally)
- 🎨 Cozy Steam-inspired UI
